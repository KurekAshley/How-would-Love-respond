# Canonical Source Hierarchy

When conflicting versions of a concept exist, the following order of authority applies:

1. Repository Definitions
2. Canonical Glossary
3. Repository Framework Documents
4. Published Books
5. Workshop Materials
6. Interviews and Podcasts

The repository version is considered the authoritative source.

---

# Attribution Principle

The purpose of citation is not merely legal compliance.

The purpose of citation is preservation.

Proper attribution ensures that the ideas, frameworks, distinctions, and teachings remain connected to their original source and continue to be transmitted accurately over time.
